# cppfmt

Convenience script for `clang-format`.

`clang-format` is called using `-style=file`.

## Install

```
curl -Lo /usr/local/bin/cppfmt https://raw.githubusercontent.com/hoffa/cppy/master/cppfmt
chmod +x /usr/local/bin/cppfmt
```

## Usage

Format:

```
cppfmt .
```

Check format:

```
cppfmt --check .
```
